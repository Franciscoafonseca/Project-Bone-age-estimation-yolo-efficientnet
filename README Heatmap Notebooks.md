# README — Heatmap Visualization Notebooks for Age Assessment Model

These notebooks generate **Grad-CAM heatmaps** for different models trained for binary age assessment.  
The code loads the trained model, processes image samples, runs Grad-CAM multiple times per image, and saves the resulting heatmaps.

---

## 📂 Project Structure

Your project directory must be organized as follows:
———————————————————————————————————————————————————————————————————-
For the notebooks to run correctly, your project directory must be organized as follows:

```bash
project-root/
│
├── data/
│ ├── FCT2025_amostra_VF.xlsx
│ ├── preprocessed/
│ │ └── clahe_center_crop_then_crop/
│
├── results/
│ ├── age10_efficientnet_v2_m.pt
│ └── other models …
│
├── Heatmaps/
│ └── (heatmaps generated by the notebook)
│
└── Heatmaps_notebook.ipynb
```

---

## 📁 Required Files

The notebook expects the following files and folders:

- `data/FCT2025_amostra_VF.xlsx` — annotation file  
- `data/preprocessed/clahe_center_crop_then_crop/` — folder containing preprocessed images  
- `results/` — folder containing the saved model checkpoints  
- `Heatmaps/` — output folder for heatmaps (created automatically if missing)

## How to Run 

1. Ensure your folder structure matches the layout above.

2. Open the notebook.

3. Run all cells in order.

4. Heatmaps will be saved in the results folder.
